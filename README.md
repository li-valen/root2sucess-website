# Root2Success Website (v3)

This repository hosts the third iteration of the Root2Success website, a platform dedicated to the Philadelphia-based math enrichment program.

## Overview

This version of the website represents a significant upgrade in design and user experience, built to better reflect the program's mission of building confidence and curiosity in mathematics.

**Key Features:**
*   **Modern Tech Stack**: Built with React, TypeScript, and Vite for high performance and maintainability.
*   **Refined UI/UX**: Features a clean, professional aesthetic with smooth scroll animations (using Framer Motion) and a responsive layout.
*   **Dynamic Gallery**: A custom masonry-style gallery that showcases community moments without cropping or distortion.
*   **Interactive Elements**: Includes animated text reveals, parallax backgrounds, and smooth navigation.
*   **Content-First**: Structured to highlight the program's impact, offerings, and student success stories.

## Project Structure

The main application code is located in the `app/` directory.

*   `app/src/components`: Reusable UI components and section-specific layouts.
*   `app/src/assets`: Images and static resources.
*   `app/src/pages`: Top-level page components.

## Tech Stack

*   **Framework**: React
*   **Build Tool**: Vite
*   **Styling**: Tailwind CSS
*   **Animations**: Framer Motion
*   **Language**: TypeScript
